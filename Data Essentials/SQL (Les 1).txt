	1. Installatie SQL
	Laat bij Instance Configuration de standaardnaam SQLEXPRESS staan
	Bij Database Engine Configuration: kies Mixed Mode en stel het wachtwoord in: digital@PXL
	
	5. Installatie controleren
	Open een command prompt via Windows + R → typ cmd → druk op Enter.
	Voer in:
	sqlcmd -?

	6. Samenvatting
	Installeer SQL Server Express met "Mixed Mode" en wachtwoord digital@PXL
	Installeer SQLCMD als opdrachtregeltool
	Test of alles werkt via sqlcmd -?
  -----------------	

	3. Verbinden met SQL Server via SQLCMD
	Met de optie -S (hoofdletter!) kan je de naam van de server meegeven waarnaar je wil verbinden. 
	En met de optie -U (ook hoofdletter!) geef je de gebruiker mee. 
	Het wachtwoord geef je tenslotte mee achter de -P optie. 
	Tijdens de cursus werken we met de sa (system administrator) user.
	Gebruik nu volgend commando om te verbinden met de server:

	sqlcmd -S .\SQLEXPRESS -U sa -P digital@PXL

	oef:
	tabel = medewerkers
	kolommen = naam, voornaam, maandsalaris, maansalaris + commissie , (maandsalaris+commissie) voor 12 maanden

	SELECT naam
		, voorn
		, maandsal
		, maansal+comm AS [Incl. Comm] 
		, (maandsal+comm)*12 AS [Totaal incl.Comm]
	FROM medewerkers;
	GO

	SELECT naam

	FROM medewerkers;
	GO

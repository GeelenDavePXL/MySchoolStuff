3. Verbinden met SQL Server via SQLCMD
Met de optie -S (hoofdletter!) kan je de naam van de server meegeven waarnaar je wil verbinden.
En met de optie -U (ook hoofdletter!) geef je de gebruiker mee. 
Het wachtwoord geef je tenslotte mee achter de -P optie.
	Tijdens de cursus werken we met de sa (system administrator) user.
	Gebruik nu volgend commando om te verbinden met de server:
sqlcmd -S .\SQLEXPRESS -U sa -P digital@PXL
